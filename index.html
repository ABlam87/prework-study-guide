HTML

  • The Head element is mostly invisible to the user, instead communicating vital info about the page with the browser
  • The Body element is what the user will be able to see
  • The Body is a 'parent' element, in that it contains all of the separate visible elements of the page
  • This list is a 'child' element of the parent 'body'

CSS

    • A margin indicates how much space we want around the outside of an element.
    • A padding indicates how much space we want around the content inside an element.
    • Element rules is the CSS style doc will apply to all elements of that type
    • Rules you only want to apply to specific elements can be done using class attributes
    • To apply rules to EVERYTHING on a page can be done with a * symbol

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: Switch to the main branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • We write functions into our js file, naing them and then adding in culry bracket the code that they'll run
  • However the site won't run this function until we specifically call it in the js file, keeping control flow in mind
